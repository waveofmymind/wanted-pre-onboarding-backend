안녕하세요! 원티드 프리온보딩 인턴십에 지원한 전상준입니다.

깃허브 링크: https://github.com/waveofmymind/wanted-pre-onboarding-backend

유저 API

POST /users/join : 회원가입

POST /users/login : 로그인

게시글 API

POST /articles : 게시글 작성(제목,내용 모두 필요)

GET /articles/{id} : 게시글 상세 조회

PUT /articles/{id} : 게시글 수정(제목,내용 모두 필요)

DELETE /articles/{id} : 게시글 삭제

GET /articles : 게시글 목록 조회(쿼리 스트링 page,size로 페이지네이션 지원, 미입력시 디폴트값 page = 0, size = 10)
